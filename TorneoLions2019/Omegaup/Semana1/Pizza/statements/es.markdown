# Descripción

A Pablo le gusta mucho la pizza, a tal grado que cada vez que se junta con amigos piden pizza. Pero siempre le toca que no se puede repartir la pizza igualmente entre todas las personas, así que Pablo se queda con las piezas que sobran.

Dado el número de piezas y el número de personas, determina cuántas piezas le tocarán a Pablo, suponiendo que cada persona come la misma cantidad de piezas (excepto Pablo al final).

# Entrada

Se darán dos números, el número de piezas **N** y el número de personas **M**, y siempre habra suficientes piezas para todos.

# Salida

El número de piezas que le tocarán a Pablo

# Ejemplo

||input
8 7
||output
2
||end

En este caso, las primeras 7 piezas se reparten entre todos y acaba sobrando una que se suma al que tenia Pablo.

# Límites

$1 \le M \le N \le 1,000,000$
